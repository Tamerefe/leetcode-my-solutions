# LeetCode 2621 – Sleep

## Problem
[LeetCode #2621 – Sleep](https://leetcode.com/problems/sleep/)

## Language
JavaScript

## Approach
Create a Promise that resolves after a specified delay using `setTimeout`.

The `sleep` function returns a Promise that resolves after `millis` milliseconds.

- Time: O(1)
